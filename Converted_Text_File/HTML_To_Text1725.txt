? prevC++ Math acosh()The function computes the arc hyperbolic cosine of an angle given in radian.Where, an arc hyperbolic cosine is the inverse operation of hyperbolic cosine.cosh-1x = acosh(x);SyntaxSuppose an angle is 'x':float acosh(float x);
double acosh(double x);
long double acosh(long double x);
double acosh(integral x);Parameterx: The value whose arc hyperbolic cosine is to be computed.Return valueIt returns the arc hyperbolic cosine of x.Example 1Let's see the simple example when the value of degree is integer type.#include
