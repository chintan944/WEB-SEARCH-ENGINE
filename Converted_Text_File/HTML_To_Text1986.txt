A vector stores the elements in contiguous memory locations and allocates the memory as needed at run time.Difference between vector and arrayAn array follows static approach, means its size cannot be changed during run time while vector implements dynamic array means it automatically resizes itself when appending elements.SyntaxConsider a vector 'v1'.
