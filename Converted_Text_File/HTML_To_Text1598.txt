CouchDB provides cURL utility to create databases. You can also use Futon the web interface of CouchDB.Creating a database using FauxtonOpen the following link in the web browser:http://127.0.0.1:5984/_utils/.You will get a page like this:Click on the "Create Database" tab in the red circle to create a database named "employees".It will show a message that database is created successfully. You can check the created database in the database tab.CouchDB Create Database using cURL UtilityUse the following syntax to create a database in CouchDB by sending an HTTP request to the server using PUT method through cURL utility.Syntax:curl -X PUT http://127.0.0.1:5984/database_nameExample:Create a database named "javatpoint". As the response the server will return you a JSON document with content "ok" : true.
