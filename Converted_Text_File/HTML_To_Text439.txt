The generated result of expression must have identical types. However, if any sub-expression is null, the generated expression is also null.Example of CONCAT FunctionIn this example, we concatenate the first two fields of each tuple.Steps to execute CONCAT FunctionCreate a text file in your local machine and insert the list of tuples.$ nano evalconcat.txtCheck the tuples inserted in the text files.$ cat evalconcat.txtUpload the text files on HDFS in the specific directory.$ hdfs dfs -put evalconcat.txt /pigexampleOpen the pig MapReduce run mode.$ pigLoad the file that contains the data.grunt> A = LOAD '/pigexample/evalconcat.txt' USING PigStorage(',') AS (a1:chararray,a2:chararray,a3:int);Now, execute and verify the data.grunt> DUMP A;Let's return the concatenation of two fields of each tuple.Result = FOREACH A GENERATE CONCAT (a1,'_',a2);
