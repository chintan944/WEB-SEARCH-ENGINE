If the return type is float, the value rounded to an integer. However, if the return type is double, the value rounded to a long.SyntaxROUND(expression)Example of ROUND FunctionIn this example, we return the round up value of the given expression.Steps to execute ROUND FunctionCreate a text file in your local machine and insert the values.$ nano mathround.txtCheck the values inserted in the text files.$ cat mathround.txtUpload the text files on HDFS in the specific directory.$ hdfs dfs -put mathround.txt /pigexampleOpen the pig MapReduce run mode.$ pigLoad the file that contains the data.grunt> A = LOAD '/pigexample/mathround.txt' AS (a1:float) ;Now, execute and verify the data.grunt> DUMP A;Let's return the round up value.grunt> Result = FOREACH A GENERATE ROUND(a1);
