It requires a preceding GROUP ALL statement for global maximums and a GROUP BY statement for group maximums. However, it ignores the NULL values.SyntaxMAX(exp)Here,exp - It is an expression with data types like chararray, int, float, long, etc.Example of MAX FunctionIn this example, we will find out the maximum of the given values.Steps to execute MAX FunctionCreate a text file in your local machine and insert the list of tuples.$ nano evalmax.txtCheck the tuples inserted in the text files.$ cat evalmax.txtUpload the text files on HDFS in the specific directory.$ hdfs dfs -put evalmax.txt /pigexampleOpen the pig MapReduce run mode.$ pigLoad the file that contains the data.grunt> A = LOAD '/pigexample/evalmax.txt' USING PigStorage(',') AS (a1:chararray,a2:chararray,a3:float) ;Now, execute and verify the data.grunt> DUMP A;Let's group the data on the basis of 'a1' field.grunt> B = GROUP A BY a1; 
grunt> DUMP B;Let's return the maximum of given numeric values.grunt> Result = FOREACH B GENERATE group, MAX(A.a3);
