If keyspace does not exist in the Cassandra, Cassandra will return an error unless IF EXISTS is used.Syntax:DROP  keyspace KeyspaceName ;Example:Let's take an example to drop the keyspace named "javatpoint".DROP keyspace javatpoint; Verification:After the execution of the above command the keyspace "javatpoint" is dropped from Cassandra with all the data and schema.You can verify it by using the "USE" command.Now you can see that "javatpoint" keyspace is dropped.
