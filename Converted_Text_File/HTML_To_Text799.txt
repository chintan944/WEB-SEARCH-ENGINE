Select the IAM Role as S3AdminAccess and keep other settings as default.Bash scripts are added in an Advanced Details.Add Storage to your instance. Root is a default volume, and I am not adding new volume.Click on the Next.Add a tag.Configure Security group.The above screen shows that we created a new security group whose name is WebServer.Review and Launch.Select an existing key pair or create a new key pair.
