All the files are located in WEB-INF/lib folder of the created WAR.classesThis element specifies a fileset of class files. After creating war, all the classes are located into WEB-INF/classes folder.webinfThis nested element specifies a fileset. All the files are located into WEB-INF directory.Apache Ant War Task Example<project name="java-ant project" default="run">
