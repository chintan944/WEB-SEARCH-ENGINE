? prevC++ Continue StatementThe C++ continue statement is used to continue loop. It continues the current flow of the program and skips the remaining code at specified condition. In case of inner loop, it continues only inner loop.jump-statement;    
continue;   C++ Continue Statement Example#include <iostream>
using namespace std;
int main()
{
