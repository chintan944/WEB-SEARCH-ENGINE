You can show in the below diagram, such as:InventoryInventory is lists of nodes or hosts having their IP addresses, databases, servers, etc. which are need to be managed.API'sThe Ansible API's works as the transport for the public or private cloud services.ModulesAnsible connected the nodes and spread out the Ansible modules programs. Ansible executes the modules and removed after finished. These modules can reside on any machine; no database or servers are required here. You can work with the chose text editor or a terminal or version control system to keep track of the changes in the content.PluginsPlugins is a piece of code that expends the core functionality of Ansible. These servers are hosted on the internet and storing the data remotely rather than the local server.
